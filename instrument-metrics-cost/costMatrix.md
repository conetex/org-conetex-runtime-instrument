|---------------------------|-------------------------------------------------------------------------------------|----------------------------------------------------------|------------------------------------------------|-------|
| **MethodCall**            | `INVOKEVIRTUAL`, `INVOKESPECIAL`, `<init>`                                         | 5                                                        | 5                                              | CPU: frame setup, Memory: parameter passing & stack |
| **MethodEntry**           | Method entry                                                                       | 5                                                        | 5                                              | CPU: setup, Memory: stack frame allocation |
| **Jump**                  | `GOTO`, `JSR`, `TABLESWITCH`, `LOOKUPSWITCH`                                      | 2                                                        | 0                                              | Branch cost, possible pipeline flush |
| **CompareInt/Object/Long**| `IF_ICMP*`, `IFNULL`, `IF_ACMP*`, `LCMP`, `FCMP*`, `DCMP*`                         | 1                                                        | 0                                              | Very cheap, register comparisons |
| **VariableLoad/Store**    | `ILOAD`, `ISTORE`, `ALOAD`, `ASTORE`                                               | 0                                                        | 1                                              | Local variable array access |
| **ArithmeticAddSubNeg**   | `IADD`, `ISUB`, `LADD`, `LSUB`, `FADD`, `FSUB`, `DADD`, `DSUB`, `INEG`             | 1                                                        | 0                                              | Extremely cheap, single ALU instruction |
| **ArithmeticMul**         | `IMUL`, `LMUL`, `FMUL`, `DMUL`                                                     | 2                                                        | 0                                              | Slightly more expensive, CPUâ€‘bound |

| **ArrayLoad/Store**       | `IALOAD`, `IASTORE`, `AALOAD`, `AASTORE`                                           | 1                                                        | 2                                              | CPU: index + bounds check, Memory: heap access |
| **ArrayNew**              | `NEWARRAY`, `ANEWARRAY`, `MULTIANEWARRAY`                                          | 2                                                        | 6                                              | CPU: setup, Memory: heap allocation |
| **FieldLoad/Store**       | `GETFIELD`, `PUTFIELD`, `GETSTATIC`, `PUTSTATIC`                                   | 1                                                        | 3                                              | CPU: decode, Memory: heap access |
| **TypeCheck**             | `CHECKCAST`, `INSTANCEOF`                                                          | 2                                                        | 1                                              | CPU: type check, Memory: class metadata |
| **Monitor**               | `MONITORENTER`, `MONITOREXIT`                                                      | 6                                                        | 4                                              | CPU: lock handling, Memory: synchronization structures |
| **ExceptionThrow**        | `ATHROW`                                                                           | 8                                                        | 4                                              | CPU: stack unwinding, Memory: new exception objects |
